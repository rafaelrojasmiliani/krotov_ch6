# Krotov's Chapter 6

This is library intended to compute Optimal Control Programs based in the Krotov's methodology introduced in the Chapter 6 of his book.

### Dependencies
    
The package relies on the GNU Scientific library 


## Compile

```bash
cd src && make
```
